# Verilog & SystemVerilog Foundation

A structured collection of digital hardware design and verification implementations built while learning **Verilog HDL** and **SystemVerilog**.

This repository documents my journey from digital design fundamentals to more advanced RTL design and verification concepts. Each module is implemented from scratch, verified through simulation, and organized by topic for easy navigation and continuous learning.
